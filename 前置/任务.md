- 工具使用：Android Studio、Git、Github 等的使用
- 开发技能：Java、ComposeUI、协程、数据库等
- 研发素养：技术文档撰写、代码规范风格考察、工程架构理解
- 业务理解：理解新闻应用的架构、核心业务流程


基础功能（必须）：
- 数据加载
- 卡片展示
进阶功能（可选）：
- 加载状态控制
- 下拉刷新和加载更多
- 数据库存储
实现要求：
- 使用 Java、ComposeUI等新技术栈实现 
- 架构采用 MVI 、MVVM等范式
其他要求
- 提交 github 项目，使用Git管理代码分支。提交项目展示录屏。
- 进度飞书文档：周报、日报
- 提交飞书技术文档，技术包含以下部分：
  - Part  1：分析实现方式
  - Part  2：UML结构
  - Part  3：主要流程图
  - Part  4：工作拆分 + 排期
- 良好的代码风格，可参照github上开源的kotlin的代码规范
